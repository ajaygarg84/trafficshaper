unsigned short  base,
char *  dest,

(bare-metal)

Parameters
numNumber which needs to be converted to float.
basenum will be divided by this base, and resultant float will be generated.
destDestination-buffer, into which the float-result will be written as string.
max_dest_sizemaximum buffer-size for dest.

There is no universal/easy solution for a bare-metal version for supporting format-specifier for float. So, this utility-function partially fulfills the requirement, as per examples below :

num = 1234, base = 1
dest will be written with 1234.0

num = 1234, base = 10
dest will be written with 123.4

num = 1234, base = 100
dest will be written with 12.34

num = 1234, base = 1000
dest will be written with 1.234
